### Estrutura

##### Javascript
```javascript
/project-root
├── /src
│   ├── /components
│   ├── /hooks
│   ├── /pages
│   ├── /styles
│   └── /utils
├── /public
├── /tests
├── package.json
└── README.md
```

##### Html, Css, Js
```html
/project-root
├── /index.html
├── /css
│ └── styles.css
├── /js
│ └── script.js
├── /images
└── README.md
```

##### Java
```java
/project-root
├── /src
│ ├── /main
│ │ ├── /java
│ │ └── /resources
│ └── /test
│ └── /java
├── pom.xml
└── README.md
```

##### Python
```python
/project-root
├── /src
│ ├── /module_name
│ │ ├── init.py
│ │ ├── main.py
│ │ └── utils.py
├── /tests
├── requirements.txt
└── README.md
```

##### Kotlin
```kotlin
/project-root
├── /src
│   ├── /main
│   │   ├── /kotlin
│   │   └── /resources
│   └── /test
│       └── /kotlin
├── build.gradle
└── README.md
```

##### C#
```c#
/project-root
├── /src
│ ├── /ProjectName
│ │ ├── Program.cs
│ │ └── /Controllers
│ └── /tests
├── ProjectName.csproj
└── README.md
```
